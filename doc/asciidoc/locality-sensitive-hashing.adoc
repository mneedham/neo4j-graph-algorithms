= One Hot Encoding

[abstract]
--
This section describes the Locality Sensitive Hashing algorithm in the Neo4j Graph Algorithms library.
--

// tag::introduction[]
The https://en.wikipedia.org/wiki/Locality-sensitive_hashing[Locality Sensitive Hashing^] algorithm is used to reduce the dimensionality of high dimensional data
It maps similar items into buckets based on a hash computation.
It can be used a pre-processing step
// end::introduction[]
